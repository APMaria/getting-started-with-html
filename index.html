Unit 1 Basics of the Web

The web is made up of HTML (Hypertest Markup Language) and hyperlinks and was invented in 1990s with over 30 billion pages.HTML is made up of:
Text Content (What you can see)
Markup (What it looks like)
References to other documents (e.g. images & video)
Links to other pages
*Note - Plain text is plain text in HTML
HTML markups are made up of tags. Some tags have opening and closing tags. Others (Void tags) do not have closing tags. In HTML all "whitespace" (e.g. multiple lines) displays as a single line. Computers are stupid and will only do exactly what you tell them to! A Tag is an HTML that is always contained within angled brackets unless it it a "Void" tag. An element refers to everything within a set of opening and closing tags. An attribute is the property of an HTML element (e.g. href).

Tags Covered
